# 010 — M2 coding: Boot flow, onboarding, Class Stone selection

Coding-expert landed M2 wiring per the dispatch in 009. Five system-level changes and five onboarding scenes comprise the boot flow up to Limbo's temporary Class Stone placeholder. (1) `CharacterProfile` schema migrated: `family: Array[String]` → `family: FamilyComposition` enum (UNDEFINED, MOTHER_ONLY, FATHER_ONLY, BOTH_PARENTS, PARENTS_AND_SIBLINGS, CHOSEN_FAMILY). Comment updated to reflect resolved 002 and the chosen-family placeholder status pending writer-led §10/§25.3 pass. (2) `RunState` adds `class_choice: ClassChoice` enum (UNDEFINED, SWORD, STAFF, SHIELD) with comment noting M3 will deliver mechanical differentiation; schema_version bumped to 2. (3) `SaveSystem` adds schema migration: loading a save with `schema_version < 2` logs warning, deletes the save, and treats as fresh-descent without crashing. (4) `RunStateRegistry` exposes `MORTALITY_TUNABLES_PATH` constant for boot scripts. (5) `SceneRouter` adds `route_to_onboarding()` (boot_picker if save exists, else content_note) and `route_to_fresh_descent()` (hard-reset RunState and route to content_note). Five new `.tscn` / `.gd` pairs under `godot/scenes/onboarding/` and `godot/scripts/onboarding/`:

**boot_picker.tscn / boot_picker.gd** — "Continue" resumes saved run, "New run" prompts confirmation, deletes save, resets state, routes to content_note. Shown only if save exists.

**content_note.tscn / content_note.gd** — Single screen rendering §26 verbatim (depression, suicidal ideation, self-harm, crisis line, "You are seen."). Header comment flags §24.2 sensitivity-read pass still owed. Single "I understand." button routes to character_create.

**character_create.tscn / character_create.gd** — Three sections: (a) Name (LineEdit, display-as-typed, Continue disabled until filled); (b) Pronouns (he/him, she/her, they/them, other with reveal-able custom field); (c) Family (preset list per resolved 002). Continue stays disabled until name ≠ empty, pronouns ≠ UNDEFINED, custom pronouns filled if selected, family ≠ UNDEFINED. Writes profile, saves, routes to class_stone.

**class_stone.tscn / class_stone.gd** — Three Polygon2D placeholder stones (Sword center default-focused per resolved 001, Staff left, Shield right). Arrow keys move focus (left/right), Enter/click commits. Confirm overlay ("Choose this path?" / "Yes" / "Cancel"). Writes class_choice, saves, routes to scene_a.

**main.gd** updated: routes fresh boot to `SceneRouter.route_to_onboarding()` instead of hardcoded path. Loads tunables via `RunStateRegistry.MORTALITY_TUNABLES_PATH` constant.

**scene_a_debug.gd** extended: F5 logs current profile (name, pronouns, pronouns_custom, family enum, class_choice enum) to console. F6 deletes save and exits to main (showing boot flow again). Input actions registered in project.godot.
